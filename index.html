Brainfuck Commands:

  • > - Move pointer right
  • < - Move pointer left
  • + - Increment cell
  • - - Decrement cell
  • . - Output cell as character
  • , - Input a character
  • [ - Start loop (if cell = 0, jump to ])
  • ] - End loop (if cell ≠ 0, jump to [)
